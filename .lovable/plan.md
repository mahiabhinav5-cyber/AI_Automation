

# AI Automation Agency Website

## Overview
A professional, modern marketing website for an AI automation agency, inspired by the clean aesthetics of Talkdesk and Teneo.ai. The site will feature a bold gradient hero, smooth scroll sections, and a polished design that communicates expertise and trustworthiness.

## Design Direction
- **Color palette**: Deep purple-to-blue gradient for hero/accents (inspired by Talkdesk), with clean white backgrounds and dark text for content sections
- **Typography**: Clean sans-serif throughout, large bold headings with lighter subtext
- **Style**: Spacious layouts, subtle animations on scroll, card-based content blocks, professional iconography

## Pages & Features

### 1. Homepage
- **Sticky navigation bar** with logo, menu links (Home, Use Cases, Testimonials, Contact), and a prominent "Request a Consultation" CTA button
- **Hero section** with gradient background, bold headline, supporting subtitle, and CTA button
- **Services overview** — 4-6 service cards (e.g., Process Automation, AI Chatbots, Data Analytics, Workflow Optimization) with icons and brief descriptions
- **Featured use cases** — 3 highlighted case study cards with industry, challenge summary, and "Read More" links
- **Client testimonials** — Carousel or grid of 3-4 testimonial quotes with names, titles, and company
- **Footer** with navigation links, contact info, and social media icons

### 2. Use Cases / Portfolio Page
- Grid of all case studies (placeholder content for ~4 projects)
- Each card shows client industry, project title, and brief summary
- Clicking a card navigates to an **individual case study page** with:
  - Client name and industry
  - The challenge
  - The solution provided
  - Results and key metrics
  - Placeholder for visuals/diagrams

### 3. Testimonials Page
- Dedicated page with all client testimonials displayed in a clean card layout
- Each testimonial includes quote, client name, title, company, and placeholder avatar/logo

### 4. Contact Page
- Clean contact form with fields: Full Name, Company Name, Email, Phone, and Message
- Client-side validation with clear error messages
- Success confirmation message on submission
- *(Frontend-only for now — no backend email sending)*

### 5. Responsive Design
- Fully responsive across desktop, tablet, and mobile
- Mobile hamburger menu navigation
- Optimized spacing and typography for all screen sizes

## Content
All content will be professional placeholder text that can be easily replaced with real agency copy, logos, and images later.
